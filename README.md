### Creating a new project

1.  Open terminal and cd to the directory you want the project to be in
2.  Run 'prosv5 conductor new-project [the name of your project]'

### Building code

1.  Open termial and cd into the project directory
2.  Run 'prosv5 make'

### Flashing code

1.  Open termial and cd into the project directory
2.  Run 'prosv5 upload'
